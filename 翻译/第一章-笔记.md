这一章是关于游戏人工智能的介绍



不同于学术型AI，游戏AI的目的只有一个，帮助开发者为玩家创造独特的体验。

 

游戏AI有许多种解决方案，常见的有

| 脚本AI               |
| -------------------- |
| 基于规则的AI         |
| 基于效用的AI         |
| 行为树               |
| 有限状态机           |
| 目标导向的AI         |
| 决策器               |
| 机器学习（很少使用） |

 

应该考虑的因素：

| **1、避免人工智障** | **避免无意义的行为  与玩家心理预期相符  与玩家恰当的沟通** |
| ------------------- | ---------------------------------------------------------- |
| 2、响应性           | 能够响应外界环境变化并采取不同行为                         |
| 3、可控性           | 能够被开发者控制，调节参数以达到想要的效果                 |
| 4、随机性           | 添加适当的随机性可以避免被轻易预测，利用                   |
| 5、简单易用         | 底层算法越简单越容易知道修改的效果                         |
| 6、可扩展性         | 对于简单情况和复杂情况都可以很好的适应                     |
| 7、小伎俩           | 可以适当的作弊，让玩家获得更好的体验                       |
